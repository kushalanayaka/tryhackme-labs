Linux is a powerful, open-source operating system widely used in cybersecurity.

Key Takeaways:

## Task 2: A Bit of Background on Linux

Linux is an open-source operating system based on Unix.
It was created by Linus Torvalds in 1991.

Key Takeaways:
- Linux is customizable and open-source
- Many distributions exist (Ubuntu, Kali Linux)

## Task 3: Interacting With Your First Linux Machine

This task introduced accessing a Linux machine through a browser-based lab.

Key Takeaways:
- Remote systems can be accessed via terminal
- No local installation is required

## Task 4: Running Your First Commands

This task introduced basic Linux commands.

Commands:
- pwd → shows current directory
- ls → lists files
- whoami → shows current user

Key Takeaways:
- Commands are essential for interacting with Linux

## Task 5: Interacting With the Filesystem

This task focused on navigating and managing files in Linux.

Commands:
- cd → change directory
- ls → list files
- touch → create file
- mkdir → create directory
- rm → delete file

Key Takeaways:
- Linux uses a hierarchical filesystem
- Navigation and file management are essential skills

## Task 6: Searching for Files

This task introduced searching for files and content in Linux.

Commands:
- find → search files
- grep → search text inside files
- locate → fast file search

Key Takeaways:
- Searching is important for finding sensitive data
- Useful in penetration testing and analysis

## Task 7: An Introduction to Shell Operators

This task introduced shell operators used to combine commands.

Operators:
- && → run next command if success
- || → run if previous fails
- | → pipe output
- > → overwrite output to file
- >> → append output

Key Takeaways:
- Operators improve efficiency
- Enable command chaining and automation
