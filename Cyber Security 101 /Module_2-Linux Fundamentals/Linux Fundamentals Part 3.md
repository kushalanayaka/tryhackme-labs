# 🐧 Linux Fundamentals Part 3

## Task 1: Introduction

This module focuses on advanced Linux concepts including permissions,
sudo usage, and file transfer utilities.

Key Takeaways:
- Builds on previous Linux fundamentals
- Introduces security-related concepts

## Task 2: Understanding Sudo

This task introduced the sudo command.

Commands Used:
- `sudo apt update`
- `sudo apt install nano`

Key Takeaways:
- sudo allows execution of commands as root
- Used for administrative tasks
- Misuse can lead to privilege escalation

## Task 3: Switching Users

This task focused on switching between users.

Commands Used:
- `su username`
- `su root`

Key Takeaways:
- su allows switching users
- Requires password
- Important for multi-user systems

## Task 4: File Permissions 

This task explored advanced permission handling.

Commands Used:
- `chmod 755 script.sh`
- `chmod +x script.sh`

Key Takeaways:
- Execute permission is required for scripts
- 755 is commonly used for executable files

## Task 5: SUID

This task introduced the SUID permission.

Commands Used:
- `chmod u+s file`

Key Takeaways:
- SUID allows a file to run with owner's privileges
- Can be dangerous if misconfigured
- Common in privilege escalation scenarios

## Task 6: Downloading Files

This task introduced file downloading in Linux.

Commands Used:
- `wget <url>`
- `curl <url>`

Key Takeaways:
- wget downloads files directly
- curl is more flexible for requests
- Useful in real-world exploitation

## Task 7: Transferring Files

This task focused on file transfer techniques.

Commands Used:
- `scp file user@host:/path`

Key Takeaways:
- Used for secure file transfer
- Important in remote systems

## Task 8: Automation Basics

This task introduced basic automation.

Commands Used:
- `nano script.sh`

Key Takeaways:
- Scripts automate repetitive tasks
- Improves efficiency
