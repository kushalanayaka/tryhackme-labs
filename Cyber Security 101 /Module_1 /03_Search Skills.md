## Task 4: Specialized Search Engines

This task introduced cybersecurity-specific search tools.

Examples:
- Shodan
  - Shodan is a specialized search engine that scans and indexes internet-connected device, allowing
    users to discover publicly exposed systems such as servers, IoT devices, webcams, routers, and industrial control systems.
- Censys
  - Censys is an internet-wide scanning and asset discovery platform that provides detailed data about hosts, services
    and SSL certificates across the internet.
- VirusTotal
  - VirusTotal is an online malware anlysis platform that scans files, Urls, IPs and hashes using multiple antivirus
    engines and threat intelligence sources
- Have I Been Pwned
  - Have I been Pwned is a data breah search that allows users to check whether their email
    addrress or password have been exposed in known data breaches

Key Takeaways:
- Specialized tools provide deeper insights
- Useful for reconnaissance and analysis
