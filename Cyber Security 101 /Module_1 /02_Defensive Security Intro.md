## Task 1: Think like a Defender
- Defensive security is the process of defending and securing the system and devices.
- This task introduced the defensive midset in cybersecurity.
- Instead of attacking the systems, defenders focus on protecting, monitoring, and responding to threats. 

 Key Takeaways:
- Assume system will be attacked
- Continues monitoring is essential
- Use layered security(defense in depth)
